# Contractor App (Next.js)

> One-line description of what this does.

## Stack
- **Next.js 16** (App Router, TypeScript)
- **Supabase** — database, auth, storage
- **Stripe** — payments
- **Resend** — transactional email
- **Vercel** — hosting

## Setup

1. Clone the repo
2. `npm install`
3. Copy `.env.example` → `.env.local` and fill in all values
4. Create a Supabase project at https://supabase.com
5. Run `npm run dev`

## Env vars required

See `.env.example` — all keys documented there.

## Deploying

Push to GitHub → connect repo in Vercel → add env vars in Vercel dashboard → deploy.
